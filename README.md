# 💎 Diamond Price Prediction — ANN Deep Learning Model

An Artificial Neural Network model that predicts diamond prices based on features like carat, cut, color, clarity, and dimensions.

## ✨ Features
- 💎 Diamond price prediction using ANN
- 📊 Feature engineering with carat, cut, color, clarity
- 🔄 Label Encoding & Standard Scaling
- ⏹️ Early Stopping implementation
- 🎯 High accuracy prediction model
